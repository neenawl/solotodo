# solotodo

A learning project — a command-line Todo List application written in Go.

## Commands

- `add {title} {text}` — add a new task
- `list` — show all tasks
- `done {title}` — mark a task as completed
- `del {title}` — delete a task
- `events` — show event history
- `help` — show available commands
- `exit` — exit the program

## Getting Started

```bash
go run main.go
```
